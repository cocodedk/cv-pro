# Backend Scripts

Utility scripts for database inspection, maintenance, and testing.

## Status

Legacy database maintenance scripts were removed after the Supabase migration.

## Supabase Alternatives

- Check local status: `supabase status`
- Reset local data: `supabase db reset`
- Export data: `supabase db dump`
- Inspect data: Supabase Studio (http://localhost:54323)
